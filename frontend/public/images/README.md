# Historical images (Phase 5)

Decorative assets for the US History Chat demo UI.

| File | Description |
|------|-------------|
| `us-flag.svg` | US flag (Wikimedia Commons, public domain) |
| `declaration.jpg` | John Trumbull, *Declaration of Independence* (1819) — Wikimedia Commons |
| `lincoln.jpg` | Abraham Lincoln portrait (O-77 matte collodion) — Wikimedia Commons |

The gallery JPGs are 200px Wikimedia thumbnails saved locally so the demo works offline without hotlinking.

Legacy SVG placeholders (`declaration.svg`, `lincoln.svg`) are kept as fallbacks but are not used by default.
